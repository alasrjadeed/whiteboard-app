# Whiteboard AL ASAR JADEED  
  
Interactive digital whiteboard for education inspired by whiteboard.fi  
  
## Features  
- Real-time collaboration  
- Room-based sessions  
- Drawing tools (Pen, Eraser, Shapes, Text)  
- Teacher dashboard  
- QR code joining  
- Image upload  
- Feedback system  
  
## Installation  
npm run install:all  
  
## Usage  
npm start (server)  
npm run client (React app)
